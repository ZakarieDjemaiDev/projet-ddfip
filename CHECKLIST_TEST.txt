## FICHE DE SECOURS

Probleme : "Erreur UnicodeEncodeError"
Solution : C'est normal sur Windows PowerShell. Les donnes sont creees,
          juste l'affichage a des soucis. Ignorez.

Probleme : "Port 5000 deja utilise"
Solution : python dashboard/app.py --port 5001
          Ou : netstat -ano | findstr :5000 (voir qui utilise)

Probleme : "Module flask not found"
Solution : pip install flask

Probleme : "Fichier data/agents.csv not found"
Solution : D'abord : python generate_data.py
